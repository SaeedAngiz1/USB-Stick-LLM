# Contributing

Thanks for helping improve USB LLM Portable Kit.

## How to contribute

1. Fork the repo
2. Create a branch (`feature/your-change`)
3. Make your updates
4. Test on at least one Windows machine
5. Open a pull request with reproduction and test notes

## Contribution focus

- Better model launcher ergonomics
- More robust hardware detection
- Additional troubleshooting scenarios
- Optional Linux/macOS portability scripts
